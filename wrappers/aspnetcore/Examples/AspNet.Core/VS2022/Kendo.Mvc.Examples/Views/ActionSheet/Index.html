Description

The ActionSheet is a dialog that presents the user with a set of alternatives to pick from. It shows on top of the app's content and must be manually dismissed before the user may resume using the app.

This demo shows how to use the Telerik UI ActionSheet for ASP.NET Core.

Functionality and Features

  • Items – option to set specific
  • TagHelper – initialization of UI components
  • API – configuration and extension points for custom functionality
  • Events – a set of client and server-side events
  • Accessibility – screen readers and keyboard support
