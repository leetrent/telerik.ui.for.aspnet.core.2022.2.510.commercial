Description

Range selection is offered out-of-the-box by the DateRangePicker control. It could also be visualized with two separate UI for ASP.NET Core DatePicker controls for a start and end date with specified minimum and maximum values. Use the client-side API and the min() and max() methods to set the value of the controls. Review the source code of the demo for complete details on the implementation.

Date range selection can be implemented by using two separate UI for ASP.NET Core DatePicker components, or simply using the built-in functionality provided by the DateRangePicker control.
