Description

The Telerik UI for ASP.NET Core Menu is a navigational component that provides multiple built-in options for:

  • Data binding
  • Menu orientation and items direction
  • Images in menu items
  • Context menu
  • Animations
  • RTL
  • Keyboard navigation and more

The demo example shows basic usage of the UI for ASP.NET Core Menu component. It visualizes data in a hierarchical multi-level menu way with unordered item lists that can be used for navigation and commands execution
