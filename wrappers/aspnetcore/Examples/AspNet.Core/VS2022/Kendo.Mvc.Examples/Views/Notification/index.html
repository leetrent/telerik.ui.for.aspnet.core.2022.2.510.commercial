Description

The Telerik UI for ASP.NET Core Notification component allows you to render various brief messages to application users. The Notification component is useful in scenarios when you need to present information related to the status of application, user action, warning or error. Key features:

  • Configuration options for positioning and stacking
  • Built-in notification types "info", "success", "warning", and "error"
  • Animated rendering
  • Automatic or configurable hiding
  • Rendering of custom content with templates
  • RTL support
