Description

This example demonstrates the cascading DropDownList functionality for ASP.NET Core. It displays three DropDownLists, where each one is enabled and filtered based on the value chosen in the previous DropDownList.

The cascading functionality is enabled after defining the CascadeFrom property, pointing to the Name of the parent DropDownList.

During initialization of the cascading functionality, the child DropDownList will:

  • Check if the CascadeFrom is defined, if not â€“ cascading will not be enabled
  • Try to find the parent object
  • Listen for changes in the parent value
  • Be disabled if the parent has no value
