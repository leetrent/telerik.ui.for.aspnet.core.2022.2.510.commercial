Description

The Telerik Core Editor supports right-to-left languages and rendering. All you have to do is to register the kendo.rtl.min.css file (located at Kendo.Mvc.Examples\Content\web installation folder) and wrap the editor in a container element with k-rtl class set for it, as demonstrated in this example.

More information on the right-to-left support of the components can be found in this topic.
