Description

The Telerik UI for ASP.NET Core Editor is a powerful WYSIWYG component, which allows you and your users to create rich text content in a familiar and user-friendly way. It provides a variety of tools for creating, editing, styling and formatting text, paragraphs, lists, images, tables, hyperlinks and other HTML elements. The component outputs identical HTML across all major browsers, follows accessibility standards and provides API for content manipulation and export to PDF.

Key Features

  • Intuitive UI for rich text editing, formatting and styling
  • Handles with ease lists, tables, images and hyperlinks creation and formatting
  • File attachment support
  • Fully customizable toolbar and buttons
  • Inline editing of different content areas
  • Format painter tool that lets to copy and paste text formatting
  • Table editing options for columns and rows merge
  • Support for predefined HTML code snippets and non-editable elements
  • Export to PDF
  • Built-in customizable themes such as Default (our own), Bootstrap and Material and more
  • Rich API for maximum flexibility
  • High performance
  • Accessibility, globalization, keyboard navigation and RTL Support
  • Rich cross-browser and mobile support
  • Responsive HTML 5 Rendering

You can find additional information on how to use the Telerik UI for ASP.NET Core Editor in this section of the product documentation."
