Description

Kendo UI TreeList for ASP.NET Core supports editing by Dragging and Dropping. When the Editable.Move property is set to true, the user can drag and drop the rows and the TreeList internally updates the parentId field.

Setting the Editable.Move.Reorderable property to true enables the nodes of the TreeList to be dragged and dropped across all levels allowing the child nodes to change also their order within the same parent scope.
