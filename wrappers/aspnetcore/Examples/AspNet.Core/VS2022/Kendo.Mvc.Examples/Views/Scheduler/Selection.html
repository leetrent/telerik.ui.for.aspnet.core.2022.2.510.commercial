Focus

  • Alt + w focuses the widget

Actions applied to the entire Scheduler

  • F10 focuses toolbar
  • B toggles view between business hours and full day (available in Day, Week and Timeline views)
  • C opens the create new event pop-up editor using the start date of the current view as a start and end
  • T navigates the view to today's time period
  • Alt + 1 - 5 switch between available views
  • Shift + Right navigates to the next time period
  • Shift + Left navigates to the previos time period

Actions applied on Scheduler toolbar

  • Right focuses next toolbar item
  • Left focuses previous toolbar item
  • Enter or Space activates toolbar item

Actions applied on Scheduler events

  • Right or Down focuses next available event
  • Left or Up focuses previous available event
  • Enter opens edit window for the selected event
  • Esc closes the edit popup window
  • Delete or Backspace opens the Delete confirmation popup to delete the event
