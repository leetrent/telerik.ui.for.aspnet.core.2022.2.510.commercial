Focus

  • Alt + w focuses the widget

Supported keys and user actions

  • Shift + Tab focus the previous item
  • Tab focuses the next item
  • Shift + Left reorder with previous item
  • Shift + Right reorder with next item
  • Ctrl + Left Decrement the width with one column
  • Ctrl + Right Increment the width with one column
  • Ctrl + Up Decrement the height with one row
  • Ctrl + Down Increment the height with one row
