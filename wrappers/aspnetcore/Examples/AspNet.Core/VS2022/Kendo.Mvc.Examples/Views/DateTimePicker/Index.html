Description

The Telerik UI for ASP.NET Core DateTimePicker is a component that allows you to manually input date and time or to select them from a dropdown calendar and a list of time options. Those features are displayed in the demo example as a basic scenario for the component. In addition, the Telerik DateTimePicker also provides the following functionalities:

  • Disabled dates
  • Selected dates
  • Start view and navigation depth
  • Validation
  • Date and time formatting
  • Calendar types
  • Week number column
  • DateInput integration
  • Templates
  • Globalization
  • Accessibility
