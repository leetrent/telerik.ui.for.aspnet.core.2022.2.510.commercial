Description

The Telerik UI for ASP.NET Core ColorGradient renders a gradient (a hue and an alpha slider) and inputs to manually enter a desired color.

Features

  • Formats—The component supports RGB and HEX formats.
  • Contrast Tool—A tool for checking the color contrast ratio between two colors.
  • Accessibility—The ColorGradient is accessible for screen readers and supports WAI-ARIA attributes.
  • Keyboard Navigation—The ColorGradient supports a number of keyboard shortcuts for processing various commands.
